# To-Do List Web App

## Overview
A simple and functional to-do list application built as part of the SkillCraft Technology web development internship (Task 4). Users can add, organize, edit, and complete tasks, along with setting a date and time for each task.

## Features
- Add new tasks to the list
- Mark tasks as completed
- Edit existing tasks
- Delete tasks
- Set a due date and time for each task
- Tasks persist using browser local storage

## Technologies Used
- HTML5
- CSS3
- JavaScript (DOM manipulation, local storage, event handling)

## How to Run
1. Clone this repository: `git clone <repo-link>`
2. Open `index.html` in your browser.
